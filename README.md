# Stone Test API

Use `/api` for endpoint documentation.

## Docker setup

```bash
$ docker compose up
```

## Non-docker setup

### Installation

```bash
$ pnpm install
```

### Running the app

```bash
# watch mode
$ pnpm start:debug
```

### Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e
```
